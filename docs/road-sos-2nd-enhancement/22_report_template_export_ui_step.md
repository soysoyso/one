# STEP 22. 보고서 템플릿 출력 UI 확장

## 목적

어드민 현장관리 화면에서 선택한 접수건을 기준으로 포트홀 관리대장을 PDF뿐 아니라 DOCX/HWPX로도 내려받을 수 있게 한다.

## 변경 범위

- 현장관리 화면 `/admin/ims/dashboard`
  - 관리대장 다운로드 영역에 출력 형식 선택 추가
  - 지원 형식: `PDF`, `DOCX`, `HWPX`
  - 기존 선택 접수건 기반 다운로드 흐름 유지
- 다운로드 API
  - 기존 `/admin/ims/report/ledger/pdf` 전용 호출 대신 `/admin/reports/export` 통합 API 사용
  - 템플릿은 `POTHOLE_LEDGER`로 전달
- QA
  - 접수관리 화면 진입 확인
  - 출력 형식 선택 옵션 확인
  - 포트홀 관리대장 DOCX/HWPX 다운로드 패키지 검증

## 검증 기준

- 관리자 로그인 후 현장관리 화면에 접근 가능해야 한다.
- 보고서 다운로드 형식 선택값이 `pdf`, `docx`, `hwpx` 순서로 제공되어야 한다.
- DOCX/HWPX 다운로드 응답은 정상 상태이며 ZIP 패키지 시그니처(`PK`)를 가져야 한다.

## 비고

- 현재 통합 출력 API에서 전체 템플릿 중 `POTHOLE_LEDGER`만 접수 선택 기반으로 직접 출력한다.
- 일상점검/상황일지는 각 기능 화면의 별도 export API로 출력한다.
- HWPX는 ZIP 기반 PoC 구조로 유지하며, 실제 한글 오피스 호환성 세부 검증은 최종 문서 단계에서 진행한다.
