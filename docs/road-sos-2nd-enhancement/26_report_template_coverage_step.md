# STEP 26. 접수 보고서 템플릿 커버리지 확장

## 목적

현장관리 접수 목록에서 선택한 접수건을 기준으로 포트홀 관리대장 외 보고서 템플릿도 기본 문서로 출력할 수 있게 한다.

## 변경 범위

- 현장관리 화면 `/admin/ims/dashboard`
  - 보고서 템플릿 선택 추가
  - 출력 형식 선택 유지
  - PDF는 포트홀 관리대장만 허용
  - 기타 템플릿 선택 시 PDF를 비활성화하고 DOCX로 전환
- 통합 보고서 API `/admin/reports/export`
  - 포트홀 관리대장 외 접수 기반 템플릿의 DOCX/HWPX 출력 허용
  - PDF는 기존 포트홀 관리대장만 유지
- 문서 생성 서비스
  - 접수 기반 기본 DOCX/HWPX 생성 메서드 추가
  - 템플릿명, 현장, 보고연도, 요약, 접수 내역 테이블 출력
- QA
  - 템플릿 선택 UI 확인
  - 사진대지 DOCX 출력 확인
  - 유지관리 결과보고서 HWPX 출력 확인
  - 포트홀 관리대장 외 PDF 요청은 400으로 거절되는지 확인

## 현재 지원 상태

| 템플릿 | DOCX | HWPX | PDF |
|---|---|---|---|
| 포트홀 관리대장 | 지원 | 지원 | 지원 |
| 포트홀 집계표 | 기본 지원 | 기본 지원 | 미지원 |
| 유지보수 일지 | 기본 지원 | 기본 지원 | 미지원 |
| 조경 작업일보 | 기본 지원 | 기본 지원 | 미지원 |
| 유지관리 결과보고서 | 기본 지원 | 기본 지원 | 미지원 |
| 사진대지 | 기본 지원 | 기본 지원 | 미지원 |

## 남은 보강

- 원본 보고서 양식의 표/결재란/사진 배치 정밀 반영
- 사진대지 실제 이미지 삽입
- HWPX 한컴 오피스 실호환성 확인
