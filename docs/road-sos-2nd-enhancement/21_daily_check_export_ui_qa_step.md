# STEP-21 일상점검 출력 UI 및 QA 보강

## 구현 범위

- 관리자 일상점검 목록/상세 화면 한글 정리
- 일상점검 상세 패널에 `DOCX 출력`, `HWPX 출력` 버튼 추가
- 일상점검 문서 출력 응답 메시지/파일명 한글 정리
- Playwright 중간 QA에 보고서 템플릿 및 상황일지 문서 출력 검증 추가

## 반영 내용

- `/admin/daily-checks` 화면의 깨진 한글 문구를 정상 한글로 교체했다.
- 상세 조회 후 선택된 `checkId` 기준으로 `/admin/daily-checks/export`를 호출할 수 있게 했다.
- QA 스크립트가 다음 항목을 추가 검증한다.
  - 일상점검 출력 버튼 존재
  - 상황일지 출력 버튼 존재
  - 보고서 템플릿 목록 9종 이상 조회
  - 상황일지 DOCX/HWPX 다운로드 및 ZIP 패키지 서명 확인
