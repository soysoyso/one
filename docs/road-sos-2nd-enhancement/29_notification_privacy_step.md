# STEP 29. 알림톡 수신자 개인정보 표시 보강

## 목적

알림톡 수신자 목록에서 휴대폰 번호가 전체 노출되지 않도록 마스킹 표시를 적용한다.

## 변경 범위

- 관리자 알림톡 수신자 설정 화면
  - 목록 휴대폰 번호 마스킹
  - 예: `010-9876-1234` -> `010-****-1234`
  - 상세/수정 패널은 원본 번호를 유지해 수정 가능하게 처리
- QA
  - 테스트 수신자 생성
  - 목록 검색 후 마스킹 표시 확인
  - 원본 번호가 목록에 노출되지 않는지 확인
  - 테스트 수신자 삭제

## 정책 연결

- `NOTIFY-002`: 전화번호는 화면 표시와 저장 방식에서 개인정보 보호를 검토한다.

## 검증

- `node qa/road_sos_mid_qa.js`
  - `notification recipient create`
  - `notification phone masked`
  - `notification recipient delete`
